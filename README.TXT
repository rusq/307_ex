0. Дистрибутив и установка
~~~~~~~~~~~~~~~~~~~~~~~~~~
Файлы, включённые в дистрибутив:

INSTALL.COM	- файл установки
MSAVT.COM	- см. "1. Программные компоненты"
SERVICE.COM	- см. "1. Программные компоненты"
307_EX.COM	- см. "1. Программные компоненты"
307_EX.TXT	- см. "2. Файл Данных"
307_EX.DAT	- появляется после использования SERVICE.COM
307_EX.RAW	- файл заставки

Установка осуществляется следующим образом:

1. Вставьте дискету в дисковод
2. Запустите INSTALL.COM с дискеты

Программа сама установит требуемые файлы и запустит программу 
обслуживания.


1. Программные компоненты
~~~~~~~~~~~~~~~~~~~~~~~~~
Данный комплекс состоит из 4-х программ:
MSAVT.COM   - резидентный шифратор/дешифратор (системная программа)
INSTALL.COM - программа-оболочка для установки и настройки
SERVICE.COM - системная программа обслуживания теста. Для запуска используется
              INSTALL.COM
307_EX.COM  - программа тестирования

Связи компонентов показаны в таблице 1. В колонке "Связан с" показаны
программы, без которых "Компонент" не может функционировать.
                   ╔═════════════╤════════════╗
                   ║  Компонент  │  Связан с  ║
                   ╠═════════════╪════════════╣
                   ║ MSAVT.COM   │ --------   ║
                   ╟─────────────┼────────────╢
                   ║ INSTALL.COM │ MSAVT.COM  ║
                   ║             │ 307_EX.COM ║
                   ║             │ SERVICE.COM║
                   ╟─────────────┼────────────╢
                   ║ 307_EX.COM  │ MSAVT.COM  ║
                   ╚═════════════╧════════════╝
                            Таблица 1

Модуль MSAVT.COM должен загружаться перед использованием как сервисной
программы, так и программы тестирования. После загрузки объём свободной 
памяти сократится максимум на 288 байт.


2. Файл данных
~~~~~~~~~~~~~~
Также, для работы комплекса необходим файл 307_EX.TXT с текстом теста. 
Структура файла должна быть следующей:

\Вопрос 1
N
1. Ответ 1
@2. Ответ 2
3. Ответ 3
  . . .
N. Ответ N
\Вопрос 2
K
1. Ответ 1
  . . .
K. Ответ K
  . . .
  и т.д.

   В первой строке указывается вопрос (отмечается знаком '\')
   Во второй строке указывается количество ответов (N)
   Следующие N строк - варианты ответов. Правильные ответы должны отмечаться 
знаком '@', неправильные ответы никак не отмечаются.
   И так далее.

Размер файла ограничен 53340 байтами: 
[65536 - (11940 + 256)] = 53340
[Размер_сегмента - (Размер_программы + PSP)] = размер_буфера_для_теста

   После создания файла данных, необходимо его зашифровать, для получения 
файла 307_EX.DAT, который должен помещатся в каталог с программой 
307_EX.COM. Больше в каталоге не должно быть ни одной программы.

3. Использование INSTALL.COM
~~~~~~~~~~~~~~~~~~~~~~~~~~~~
ПРОГРАММА ОБСЛУЖИВАНИЯ ТЕСТА ПРЕДНАЗНАЧЕНА ТОЛЬКО ДЛЯ ПРЕПОДАВАТЕЛЯ И НЕ 
ДОЛЖНА ПОПАДАТЬ В РУКИ К СТУДЕНТАМ

Запуск программы осуществляется на нужном компьютере с дискеты

После запуска программы появляется меню для выбора операции.
╔══════════════╗
║ Обновить     ║
║ Расшифровать ║
║ Зашифровать  ║
║ Настроить    ║
║ Отмена       ║
╚══════════════╝

Пункт "Обновить" позволяет копировать обновлённые файлы с дискеты на 
компьютер. (Если вы заранее зашифровали файл и сделали необходимые 
настройки, просто выбирайте этот пункт)

Второй пункт предназначен для расшифровки файла данных 307_EX.DAT в 
текстовый файл 307_EX.TXT. Для расшифровки требуется знание пароля.

При выборе третьего пункта производится шифрование файла 307_EX.TXT в
файл данных 307_EX.DAT. Для шифрования предлагается ввести пароль из 8-ми 
символов.

Пункт "Настроить" предлагает изменить количество вопросов выдаваемых 
тестом, и количество правильных ответов для успешной сдачи теста. По 
умолчанию выдаётся 7 вопросов, из которых на 5 нужно дать правильный ответ 
для успешной сдачи.

Пункт "Отмена" служит для выхода из программы. Также можно выйти, нажав 
клавишу [Esc]. При этом файлы на жестком диске не будут обновлены.


4. Использование 307_EX.COM
~~~~~~~~~~~~~~~~~~~~~~~~~~~
Данная программа является оболочкой для тестирования студентов.

При запуске на компьютерах, оборудованных VGA, выдается красочная заставка.

Ход выполнения можно условно разделить на четыре этапа:

1. Памятка студенту
   На экране выводятся сведения о клавишах, используемых в программе.

2. Ввод фамилии
   Здесь следует вводить фамилию. Длина фамилии не может быть больше 19, 
   или меньше двух. Принимаются только русские прописные и строчные буквы.

3. Тестирование
   На данном этапе задаётся столько вопросов, сколько было установлено с 
   помощью программы установки.

4. Оценка результатов
   На экране выводятся фамилия, время и результат. В том случае, если 
   результат положительный - фон экрана зелёный, в противном случае - 
   красный. Выход осуществляется нажатием комбинации клавиш [Ctrl]+[H]
   (английская).

При переходе на четвёртый, заключительный этап, издаётся звуковой сигнал.

Системная программа. Загружается автоматически. Отвечает за корректную шифровку
и дешифровку данных.

6. Описание SERVICE.COM
~~~~~~~~~~~~~~~~~~~~~~~
Системная программа обслуживания. Запускается из INSTALL.COM. Отдельно не 
работает.

7. Системные требования
~~~~~~~~~~~~~~~~~~~~~~~
IBM PC совместимый компьютер (с монитором и клавиатурой :) )
286+ процессор
EGA или VGA адаптер

MSDOS версии 2.0 и выше
100 Кб свободного места на диске
64 Кб оперативной памяти

ПРОГРАММА НЕ РАБОТАЕТ ПОД MS WINDOWS NT!!! Вернее работает, но криво :) !
